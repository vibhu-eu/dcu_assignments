# dcu_assignments
Data Analytics and Story Telling Assignment#2 Repo

The Assignment #2 is to create a Quarto Report as an HTML file an upload to Gihub pages. The raw csv files given to this assignment are unicef_indicator1.csv, unicef_indicator2.csv and unicef_metadata.csv

Since data for selevtive year and selective country was given, several assumptions were made and NaN was used to compensate fo rmissing values.

The processing was done using polar libraries (Pandas library was used for the Tableau Asssignment).
The final Quarto Report in HTML file has been uploaded with story and graphics generated in Google Colab.
